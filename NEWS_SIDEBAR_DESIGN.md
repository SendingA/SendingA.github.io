## ✨ 主要特性

### 1. **左侧固定日历栏**
- 固定宽度 280px
- 粘性定位（滚动时保持可见）
- 紧凑的3列月份网格
- 实时显示筛选状态和数量
- Reset按钮快速清除筛选

### 2. **右侧新闻主内容区**
- 自适应宽度
- 新闻时间线展示
- 底部分页导航
- 显示页码和条目统计

### 3. **智能筛选**
- 点击月份 → 查看该月新闻
- 展开日期选择器 → 精确到某天
- 自动更新计数和标题
- 没有新闻时显示提示信息

### 4. **响应式适配**
- **桌面端 (>1024px)**: 左右分栏，日历固定左侧
- **平板端 (768px-1024px)**: 缩小日历宽度
- **手机端 (<768px)**: 日历移到底部，单列布局

## 📏 设计规格

### 左侧日历栏
- **宽度**: 280px (桌面), 250px (平板)
- **背景**: 白色卡片
- **阴影**: 0 4px 20px rgba(0,0,0,0.08)
- **圆角**: 15px
- **内边距**: 1.5rem

### 月份按钮
- **网格**: 3列
- **尺寸**: 自适应
- **字体**: 0.75rem
- **颜色**: 渐变紫 (#667eea → #764ba2)
- **选中**: 绿色渐变 (#28a745 → #20c997)

### 分页样式
- **按钮样式**: 圆角卡片
- **激活状态**: 渐变紫色背景
- **悬停效果**: 上浮 + 阴影
- **信息显示**: 页码 / 总页数 (条目数)

## 🎯 使用说明

### 查看新闻
1. 打开 `/news` 页面
2. 默认显示最新的新闻（分页显示）
3. 使用底部分页导航浏览更多

### 按月份筛选
1. 左侧找到对应年份
2. 点击有新闻的月份（有数字徽章）
3. 右侧自动显示该月新闻
4. 日期选择器自动展开

### 按日期筛选
1. 先选择月份
2. 日期选择器中点击具体日期
3. 只显示该日期的新闻

### 清除筛选
- 点击左侧的 "Reset" 按钮
- 或再次点击已选中的月份

### 分页浏览
- 使用底部的 `« 1 2 3 »` 导航
- 查看 "Page X of Y (N items)" 了解当前位置

## ⚙️ 配置

### 设置每页显示数量

编辑 `config/_default/hugo.yaml`:
```yaml
paginate: 6  # 每页显示6条新闻
```

或使用其他数字，如：
- `paginate: 5` - 每页5条
- `paginate: 10` - 每页10条
- `paginate: 8` - 每页8条

### 修改日历栏宽度

编辑 `layouts/section/news.html` 的CSS部分:
```css
.news-layout-container {
  grid-template-columns: 300px 1fr;  /* 改为300px */
  gap: 2rem;
}
```

### 修改主题颜色

```css
/* 主色 */
background: linear-gradient(135deg, #YOUR_COLOR_1 0%, #YOUR_COLOR_2 100%);

/* 选中色 */
background: linear-gradient(135deg, #28a745 0%, #20c997 100%);
```

## 📱 响应式行为

### 桌面 (>1024px)
- 左侧日历固定 280px
- 右侧新闻占据剩余空间
- 日历随页面滚动固定显示

### 平板 (768px-1024px)
- 日历宽度缩小到 250px
- 保持左右分栏布局
- 月份网格保持3列

### 手机 (<768px)
- **单列布局**
- 新闻显示在上方
- 日历移到下方
- 月份网格改为4列
- 日期网格改为5列
